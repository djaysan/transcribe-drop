Transcribe a video, or follow a channel — all on your Mac.

or drop a video onto the app icon in your Dock
These are transcribed one by one (Facebook has no captions, so it uses on-device Whisper). Following a whole Facebook profile isn't possible — its reels tab can't be listed — so paste the reels you want here.
Copy the prompt below into Claude (or Claude Code), replace the [brackets] with a couple of sentences about yourself, and save the reply as context.md in this app's folder. Then the summaries will flag which videos matter to you.
Both the key and context file are stored locally in settings.json (gitignored, never uploaded). The context file is optional — without it you still get plain summaries.
